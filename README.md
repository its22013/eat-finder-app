## 概要
このアプリは、外食先を選ぶ際に「どこに行こうか」と悩むユーザーをサポートするために開発されました。  
ホットペッパーグルメAPIを利用して近隣の飲食店情報を取得し、ルーレット形式でランダムに1店舗を提案する機能を備えています。  
さらに、お気に入りの飲食店だけでルーレットをしてランダムで飲食店をきめる機能もあります。  
また、スマートフォンでも快適に利用できるよう、操作性やデザインにも配慮しています。
